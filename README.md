# Smart-Energy-Copilot-v2.0-Multimodal-Edge-AI-Dashboard
A multimodal AI-powered “Smart Energy Copilot” dashboard that uses PaddleOCR-VL to parse bills and IoT logs, ERNIE fine-tuned via Unsloth to interpret energy patterns, and CAMEL-AI agents to generate actionable insights. Deployed on RDK X5 for private, low-latency edge use, with sleek ERNIE-built web visualization.
